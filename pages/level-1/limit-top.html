LIMIT / TOP – Restrict Number of Results

Use this to fetch only a specific number of rows.

When do we use LIMIT or TOP?

We use these keywords to retrieve just a small portion of rows from a query result — often for previews, pagination, or top-ranked lists.

  • Show only top 5 recent orders
  • Preview first 10 customers
  • Limit rows for performance testing

Syntax


SELECT TOP 5 * 
FROM Customers;


SELECT * 
FROM Customers
LIMIT 5;

Example 1 – First 3 Customers

SELECT TOP 3 CustomerID, CompanyName
FROM Customers;

SELECT TOP 2 OrderID, OrderDate
FROM Orders
