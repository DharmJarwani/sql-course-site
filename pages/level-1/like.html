LIKE – Match Text Patterns

Find records using wildcard patterns like % and _.

When do we use LIKE?

LIKE is used to filter rows where a column's value matches a pattern. Common use cases include:

  • Search names starting with 'A'
  • Find emails ending in "@gmail.com"
  • Match partial phone numbers or addresses

LIKE Syntax

SELECT * 
FROM Customers
WHERE CustomerName LIKE 'A%';

Example 1 – Names Starting with "A"

SELECT CustomerID, CompanyName
FROM Customers
WHERE CompanyName LIKE 'A%';

Example 2 – Emails Ending with @gmail.com

SELECT ContactName, Email
FROM Customers
WHERE Email LIKE '%@gmail.com';
