IS NULL – Find Missing Data

Used to filter rows where a column has no value (NULL).

When do we use IS NULL?

NULL means missing or unknown data. You use IS NULL or IS NOT NULL to filter rows based on whether a field is blank (null) or filled.

  • Find orders with no shipping date
  • Detect customers missing contact info
  • Clean incomplete or partial records

NULL Check Syntax

SELECT *
FROM Orders
WHERE ShippedDate IS NULL;

Example 1 – Orders Not Yet Shipped

SELECT OrderID, CustomerID, ShippedDate
FROM Orders
WHERE ShippedDate IS NULL;
