WHERE – Filter Rows With Conditions

WHERE clause se table ke andar sirf wahi rows nikalte hain jo given condition ko satisfy karti हैं.

WHERE कब काम आता है?

  • Country = 'USA' वाले customers की लिस्ट निकालना
  • OrderAmount > 10,000 के high-value orders track करना
  • Date range filter करके monthly sales report बनाना

SQL Server Query

SELECT OrderID, CustomerID, OrderDate
FROM   Orders
WHERE  ShipCountry = 'USA';
