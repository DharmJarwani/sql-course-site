DEFAULT – Set Automatic Values

Use DEFAULT to auto-fill columns when no value is provided.

What is the DEFAULT Constraint?

The DEFAULT constraint assigns a value to a column if no value is explicitly inserted. This is useful for:

  • Setting current date as default for created or order time
  • Predefining status like 'Pending' or 'New'
  • Ensuring non-null values even if user skips a column

DEFAULT Constraint Syntax

CREATE TABLE Employees (
  EmpID INT PRIMARY KEY,
  JoinDate DATE DEFAULT GETDATE()
);

Example – Default Order Date

ALTER TABLE Orders
ADD CONSTRAINT DF_OrderDate
DEFAULT GETDATE() FOR OrderDate;
