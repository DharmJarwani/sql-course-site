AUTO INCREMENT – Auto-generate Identity Values

Automatically generate a unique ID whenever a new row is inserted.

What is AUTO INCREMENT?

The AUTO INCREMENT feature allows a column (usually the primary key) to automatically generate a unique number whenever a new record is added. This is useful for assigning sequential IDs to rows without manually entering them.

In SQL Server, we use the IDENTITY(seed, increment) keyword to define an auto-incrementing column.

When do we use AUTO INCREMENT?

  • Automatically assigning unique IDs to new records
  • Replacing the need for manual entry of primary keys
  • Simplifying data insertions in systems like orders, customers, etc.

Auto Increment Syntax (SQL Server)

CREATE TABLE Customers (
  CustomerID INT IDENTITY(1,1) PRIMARY KEY,
  Name VARCHAR(50),
  Country VARCHAR(50)
);

Insert Without Specifying ID

INSERT INTO Customers (Name, Country)
VALUES ('Alice', 'USA');

No need to specify CustomerID — it will auto-generate the next number.
