COUNT, AVG, SUM – Aggregate Functions

Quickly summarize values in columns with totals, averages, and counts.

When do we use COUNT, AVG, and SUM?

These aggregate functions are used to compute values over a set of rows — for insights, summaries, and metrics.

  • Count total number of orders or customers
  • Get average freight, price, or quantity
  • Sum total sales or revenue

Basic Syntax

SELECT 
   COUNT(*) AS TotalCount,
   AVG(Freight) AS AverageFreight,
   SUM(Freight) AS TotalFreight
FROM Orders;

Example 1 – Total Number of Orders

SELECT COUNT(*) AS TotalOrders
FROM Orders;
TotalOrders
830

Example 2 – Average Freight Per Order

SELECT AVG(Freight) AS AvgFreight
FROM Orders;
AvgFreight
32.5

Example 3 – Total Freight to USA

SELECT SUM(Freight) AS FreightToUSA
FROM Orders
WHERE ShipCountry = 'USA';
FreightToUSA
25000.75
