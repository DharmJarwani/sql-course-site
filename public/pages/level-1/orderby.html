ORDER BY – Sort Query Results

Sort your output rows based on one or more columns.

When do we use ORDER BY?

Use ORDER BY to sort your result set either in ascending (default) or descending order. You can sort on a single column or multiple columns.

  • Alphabetical lists of names
  • Latest orders by date
  • Top-selling products by sales

Syntax

SELECT * 
FROM table_name
ORDER BY column1 ASC, column2 DESC;

Example 1 – Customers by City (A–Z)

SELECT CustomerID, City
FROM Customers
ORDER BY City ASC;

Example 2 – Orders by Date (Latest First)

SELECT OrderID, OrderDate
FROM Orders
ORDER BY OrderDate DESC;
