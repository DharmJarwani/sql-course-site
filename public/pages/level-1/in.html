IN – Match Multiple Values

Check if a value exists in a list of values.

When do we use IN?

The IN operator allows us to test if a column's value matches **any** value in a specified list. It's useful for filtering based on multiple options without writing multiple OR conditions.

  • Filter by selected countries or cities
  • Match a list of known product categories
  • Find users with specific roles or IDs

IN Syntax

SELECT * 
FROM Customers
WHERE Country IN ('Germany', 'France', 'UK');

Example 1 – Customers from Selected Countries

SELECT CustomerID, CompanyName, Country
FROM Customers
WHERE Country IN ('Germany', 'France', 'UK');
