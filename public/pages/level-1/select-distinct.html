SELECT DISTINCT – Remove Duplicates

Get unique values from a column by eliminating duplicates.

When do we use SELECT DISTINCT?

SELECT DISTINCT is used to return only different (unique) values from a column. It is helpful when you want to remove duplicates from a result set.

  • List of unique cities customers are from
  • Unique product categories sold
  • Filtering dropdown values in a UI

Basic Syntax

SELECT DISTINCT column_name
FROM table_name;

Example 1 – Unique Ship Countries

SELECT DISTINCT ShipCountry
FROM Orders;
ShipCountry
Germany
France
Brazil

Example 2 – Unique Customer Cities

SELECT DISTINCT City
FROM Customers;
City
Berlin
Madrid
London
