SELECT – Retrieve Data

Pick the columns you need from the rows you want.

When do we use SELECT?

  • Customer lists for dashboards
  • Finance / inventory reports
  • Feeding BI & analytics tools

SQL Server Query

SELECT FirstName, LastName, City
FROM   Customers
WHERE  Country = 'Germany';
