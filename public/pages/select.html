SELECT – Retrieve Data From a Table

The SELECT statement picks the columns you want, from the rows you need.

When is SELECT used?

  • Displaying customer lists in dashboards
  • Generating reports for finance or inventory
  • Feeding data into BI / analytics tools

SQL Server Query

SELECT FirstName, LastName, City
FROM   Customers
WHERE  Country = 'Germany';
