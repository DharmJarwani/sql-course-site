Wildcards – Flexible Text Matching

Use LIKE with % or _ to search patterns inside text columns.

When do we use LIKE wildcards?

The LIKE operator is used to find values that match a specific pattern, especially in text fields.

  • Search names or emails starting with certain letters
  • Find SKUs or codes that follow a fixed pattern
  • Filter records with partial values (e.g., product names)

Wildcard Syntax

SELECT * 
FROM   table_name
WHERE  column LIKE 'pattern';

Example 1 – Names starting with 'A'

SELECT FirstName, LastName
FROM   Employees
WHERE  FirstName LIKE 'A%';

Example 2 – Cities ending in 'burg'

SELECT City
FROM   Customers
WHERE  City LIKE '%burg';

Example 3 – Product names containing 'cheese'

SELECT ProductName
FROM   Products
WHERE  ProductName LIKE '%cheese%';

Example 4 – Employees with 5-letter names

SELECT FirstName
FROM   Employees
WHERE  FirstName LIKE '_____';

Example 5 – Customers from cities containing 'port'

SELECT CustomerID, City
FROM   Customers
WHERE  City LIKE '%port%';
