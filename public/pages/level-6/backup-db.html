BACKUP – Save Your Database

Use BACKUP to protect your data by creating a full copy of your database.

What is the BACKUP Command?

The BACKUP DATABASE statement creates a complete copy of your database and stores it in a specified file. It is crucial for:

  • Disaster recovery and prevention of data loss
  • Transferring databases between servers
  • Creating snapshots before major changes

Typical BACKUP DATABASE Syntax

BACKUP DATABASE Northwind
TO DISK = 'D:\Backups\Northwind.bak';

Additional Options

  • WITH FORMAT – creates a new media set and erases existing backups
  • WITH COMPRESSION – saves disk space by compressing the backup
  • WITH DIFFERENTIAL – backs up only changed data since last full backup

Example – Compressed Full Backup

BACKUP DATABASE Northwind
TO DISK = 'D:\Backups\Northwind_Full.bak'
WITH COMPRESSION, INIT;
