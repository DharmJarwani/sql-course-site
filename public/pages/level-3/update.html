UPDATE – Modify Data

Change existing values in a table using filters.

When do we use UPDATE?

The UPDATE statement is used to modify existing records in a table. You typically use it with a WHERE clause to avoid changing all rows.

  • Correcting customer information
  • Updating order status or payment info
  • Adjusting prices or stock levels

Syntax

UPDATE table_name
SET    column1 = 'new_value',
         column2 = 'new_value'
WHERE  condition;

Example – Update Country Name

UPDATE Customers
SET    Country = 'USA'
WHERE  Country = 'United States';
